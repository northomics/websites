For thermal Raw files

  • convert raw files to .mgf
    • you will be able to see .mgf files generated in the same directory of raw files
  • mgf clustering to remove spectra redundancy to speed up the search
  • first round of database search against the “big” fasta database by Xtandem
  • extract the fasta from the Xtandem search result, to generate a sample specific database
    • you will be able see this file under output_folder/database/, a string of “sample_specific” will be appended to the original fasta file name.
  • database search using the sample specific fasta database
  • taxon mapping
  • function mapping
